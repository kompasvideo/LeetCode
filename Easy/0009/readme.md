# 9. Число-палиндром
Для заданного целого числа x верните trueеслиxявляетсяпалиндромом, то falseиначе.

 

Пример 1:
>Входные данные: x = 121   
Выходные данные: true   
Пояснение: 121 читается как 121 слева направо и справа налево.

Пример 2:   
> Входные данные: x = -121   
Выходные данные: false   
Пояснение: при чтении слева направо получается -121. При чтении справа налево получается 121-. Следовательно, это не палиндром.
 
Пример 3:
> Входные данные: x = 10  
Выходные данные: false   
Пояснение: Число 01 читается справа налево. Следовательно, это не палиндром

Ограничения:
* -2³¹ <= x <= 2³¹ - 1

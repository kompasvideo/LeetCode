# 21. Объединение двух отсортированных списков
Вам даны головы двух отсортированных связанных списков list1 и list2.

Объедините два списка в один отсортированный список. Список должен состоять из объединенных узлов первых двух списков.

Верните голову объединенного связанного списка.

Пример 1:
![Graph](merge_ex1.jpg "Graph")
>Входные данные: list1 = [1,2,4], list2 = [1,3,4]   
Выходные данные: [1,1,2,3,4,4]   

Пример 2:   
> Входные данные: list1 = [], list2 = []   
Выходные данные: []   

Пример 3:   
> Входные данные: list1 = [], list2 = [0]   
Выходные данные: [0]   
 
Ограничения:
* Количество узлов в обоих списках находится в диапазоне [0, 50].
* -100 <= Node.val <= 100
* И list1 и list2 отсортированы в неубывающем порядке.
